# Copilot Capability Transfer

GitHub Copilot skills for structured thinking, writing, and analysis. Copy any skill folder into `.github/skills/` in your repo.

## Skills

| Skill | What it does |
|-------|-------------|
| **skill-creator** | Create, modify, and evaluate Copilot skills. Adapted from [Anthropic's skill-creator plugin](https://github.com/anthropics/claude-plugins-official). |
| **requirements** | Elicit and structure requirements before work begins. Covers functional requirements, quality attributes, stakeholder analysis, and failure definition. |
| **right-thing-right-way** | Two-altitude diagnostic: Am I solving the right problem? Am I solving it the right way? |
| **systems-analysis** | Structured systems analysis with data flows, failure modes, leverage points, and architecture documentation (C4 + arc42). |
| **value-analysis** | Structured value reasoning — value for whom, at what altitude, under what constraints. |
| **briefing** | Produce stakeholder briefing products using intelligence, military, and business communication standards (BLUF, SOF, options lists). |
| **editorial** | Editorial governance for human-facing content — tone/style, production standards, writing craft principles. |
| **doc-check** | Evaluate documentation against the Diataxis 4-type framework (tutorial, how-to, reference, explanation). |
| **artifact-review** | Evaluate artifacts from expert perspectives using selectable review lenses (usability, clarity, stakeholder fit). |

## Install

Copy a skill folder into your repo:

```
your-repo/.github/skills/<skill-name>/SKILL.md
```

Some skills include a `references/` subfolder — copy that too.
